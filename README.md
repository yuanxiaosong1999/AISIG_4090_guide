## 🖥️ 服务器基本信息

| 项目 | 值 |
|---|---|
| 主机名 | `tjzs-Rack-Server` |
| GPU | 8 × RTX 4090（48GB） |
| 系统 | Ubuntu 22.04 |
| SSH 端口 | **23222** |
| 访问 | 校园网 / 学校 VPN / 校外内网穿透 |

## 🚀 快速开始

```bash
# 连接服务器
ssh -p 23222 <你的用户名>@<服务器IP>
```

完整使用方法见 [服务器须知](./server_guide.md)。
